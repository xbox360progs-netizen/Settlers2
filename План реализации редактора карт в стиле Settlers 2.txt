# План реализации редактора карт в стиле Settlers 2

План по созданию полнофункционального редактора карт с поддержкой прямоугольной и изометрической проекций, управляемого геймпадом для Xbox 360.

## Этап 1: Создание констант карты

**Файл:** `Logic/MapConstants.h` (новый)

- Создать файл с предоставленными константами
- Добавить директорию Logic в проект
- Включить константы в stdafx.h для глобального доступа

Константы включают:
- Размеры тайлов (прямоугольные 238x148, изометрические 128x64)
- Размеры карты по умолчанию (20x20)
- Параметры изометрической сетки
- Параметры клиппинга ромба

## Этап 2: Система отображения тайлов

**Файл:** `Graphics/TileRenderer.h/cpp` (новый)

Создать класс для рендеринга тайлов в двух проекциях:

- **Методы:**
  - `RenderTile()` - рендеринг одного тайла
  - `RenderLayer()` - рендеринг слоя тайлов
  - `RenderMap()` - рендеринг всей карты
  - `SetProjectionMode()` - переключение между прямоугольной/изометрической
  - `WorldToScreen()` - конвертация координат
  - `ScreenToWorld()` - конвертация координат

- **Прямоугольная проекция:**
  - Тайлы 238x148 пикселей
  - Простая сетка с наложением

- **Изометрическая проекция:**
  - Тайлы 128x64 пикселей
  - Ромбовидная сетка
  - Корректное Z-ordering для перекрытия

## Этап 3: UI редактора карт

**Файл:** `UI/EditorUI.h/cpp` (новый)

Создать специализированный UI для редактора:

- **TilePalette** - палитра выбора тайлов
  - Отображение всех типов тайлов (Grass, Water, Mountain, и т.д.)
  - Навигация по палитре геймпадом
  - Подсветка выбранного тайла
  - Поддержка слоёв (Ground, Objects, Overlay)

- **EditorPanel** - панель инструментов
  - Кнопка переключения проекции (Rect/ISO)
  - Кнопка изменения размера карты
  - Кнопка сохранения/загрузки
  - Отображение текущих координат курсора

- **SizeDialog** - диалог изменения размера
  - Использование ScreenKeyboard для ввода размеров
  - Валидация значений

## Этап 4: Интеграция EditorScene

**Файл:** `Scene/EditorScene.h/cpp` (модификация)

Расширить существующий EditorScene:

- **Добавить поля:**
  - `TileRenderer* m_tileRenderer`
  - `EditorUI* m_editorUI`
  - `Camera* m_worldCamera`
  - `Camera* m_uiCamera`
  - `Input::Gamepad* m_gamepad`
  - `Input::ScreenKeyboard* m_screenKeyboard`
  - `TileType m_selectedTileType`
  - `LayerType m_selectedLayer`
  - `ProjectionMode m_projectionMode`
  - `int m_cursorX, m_cursorY` - позиция курсора на карте

- **Методы:**
  - `Initialize()` - инициализация всех подсистем
  - `HandleInput()` - обработка геймпада
  - `PaintTile()` - установка тайла
  - `ResizeMap()` - изменение размера
  - `SaveMap()` / `LoadMap()` - сохранение/загрузка
  - `ToggleProjection()` - переключение проекции

## Этап 5: Управление через геймпад

**Файл:** `Scene/EditorScene.cpp` (модификация)

Реализовать управление:

- **Навигация по карте:**
  - Левый стик - панорамирование камеры
  - Правый стик - быстрое панорамирование
  - LT/RT - зум камеры

- **Навигация по UI:**
  - D-Pad - навигация по палитре тайлов
  - A - выбор тайла / установка тайла на карту
  - B - отмена / возврат
  - X - смена слоя (Ground/Objects/Overlay)
  - Y - переключение проекции
  - Start - открыть меню
  - Back - отмена действия

- **Установка тайлов:**
  - Курсор на карте следует за камерой
  - Зажатие A для рисования кистью
  - Предпросмотр выбранного тайла под курсором

## Этап 6: Изменение размера карты

**Файл:** `Scene/EditorScene.cpp` (модификация)

Реализовать функционал:

- Открытие диалога изменения размера
- Использование ScreenKeyboard для ввода ширины/высоты
- Вызов `Map::Resize()` с новыми параметрами
- Сохранение данных при ресайзе (копирование существующих тайлов)
- Обновление TileRenderer с новыми размерами

## Этап 7: Сохранение и загрузка карты

**Файл:** `World/MapSerializer.h/cpp` (новый)

Создать систему сериализации:

- **Формат файла:** JSON или бинарный
- **Сохранение:**
  - Размеры карты
  - Все слои (Ground, Objects, Overlay)
  - Типы тайлов
  - Свойства тайлов (height, walkable, buildable)

- **Загрузка:**
  - Парсинг файла
  - Создание карты с нужными размерами
  - Заполнение слоёв данными

- **Интеграция:**
  - Методы `Map::Save()` и `Map::Load()`
  - Вызов из EditorScene через меню

## Этап 8: Рендеринг UI редактора

**Файл:** `UI/EditorUI.cpp` (новый)

Реализовать отрисовку:

- **TilePalette:**
  - Сетка с превью тайлов
  - Подсветка выбранного тайла
  - Названия слоёв

- **EditorPanel:**
  - Кнопки с иконками/текстом
  - Информация о текущем состоянии

- **Курсор на карте:**
  - Подсветка тайла под курсором
  - Превью выбранного тайла (полупрозрачный)

## Порядок реализации

1. MapConstants.h → TileRenderer → EditorUI → EditorScene интеграция → Управление геймпадом → Ресайз карты → Сериализация → UI рендеринг
